# React Calculator App
This project is a basic react calculator app. Here, React components, JSX, React states and props, and event handlers were utilized.

## Built With
- CSS
- JavaScript
- React

## Live Demo
[Live Demo](https://calculator-milestone-5.netlify.app/)

## Getting Started 
To get a local copy up and running follow these simple example steps.

### Prerequisites

Make sure Node.js is running on your local machine

### Setup

~~~bash
$ git clone https://github.com/Alaska01/Calculator-App.git
$ cd Calculator-App
~~~

Install modules:

```
yarn install
```

### Usage

Start server with:

```
    yarn start or npm start
```
